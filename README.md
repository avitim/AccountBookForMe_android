# My money

## コンセプト

私の、私による、私のための家計簿アプリ
訳: 自分がほしい機能を詰めました

## 特徴

一般的な家計簿アプリと異なり、一つの支出に複数の品物や支払いが登録できます。
品物や支払いの種類ごとに表示できるので、過去に購入した品物の金額が確認できたり、ポイントや現金併用といった複雑な支払いも記録できます。

## 機能

* 支出登録・編集・削除
  * 日付
  * 店舗
  * 品物
    * 品名
    * 金額
    * カテゴリ
  * 支払い
    * 決済方法
    * 金額
  * メモ（任意）
* カテゴリごとの総額、品物一覧表示
* 決済方法ごとの総額、支出一覧表示
* 店舗ごとの総額、支出一覧表示
* カテゴリ、決済方法、店舗の登録・編集・削除
